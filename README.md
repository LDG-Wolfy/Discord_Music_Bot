Now command 
  - ~명령어 : 사용할 수 있는 명령어 확인
  - ~입장 : 음악 봇을 사용자가 있는 음성채널로 입장
  - ~나가 : 음성채널에서 강퇴
  - ~play <Link> : Youtube 주소를 가져와 바로 재생 
  - ~pause : 재생중인 음악 일시정지
  - ~stop : 재생중인 음악 종료
  - ~선택 <A, B, C> : 선택하기 어려울때 사용
